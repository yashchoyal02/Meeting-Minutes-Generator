# AI-Powered Meeting Transcription & Minutes Generation System

## Overview

An automated system that converts meeting audio recordings into professionally formatted Minutes of Meeting documents.

## Features

- Speaker diarization using pyannote.audio
- Speech-to-text using OpenAI Whisper
- AI-powered meeting summarization using Gemini AI
- Action item extraction
- PDF report generation
- Multi-speaker support

## Architecture

Audio
→ Speaker Diarization
→ Transcription
→ Summarization
→ PDF Generation

## Tech Stack

- Python
- Whisper
- pyannote.audio
- Gemini AI
- ReportLab

## Results

Reduced meeting documentation time from 3–5 hours to under 2 minutes.
